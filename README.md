# Sudeeksha Convention Centre

## Project Info

Official website for **Sudeeksha Convention Centre**, a premier wedding and events venue located at Magadi Main Road, Bengaluru.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Local Development

**Requirements:** Node.js & npm

```sh
# Step 1: Clone the repository
git clone https://github.com/coastalwebdeveloper/Sudeeksha-Website.git

# Step 2: Navigate to the project directory
cd Sudeeksha-Website

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```
